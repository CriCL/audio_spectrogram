# audio spectrogram
Shows a live spectrogram streaming using a microphone

## Installation
uses Python 3+

#### Libraries
To make this project works, you have to install the following libraries using:

> pip install -r requirements.txt


| library | version    |
| -----   | --------   |
| pyaudio   | **0.2.11** |
| numpy   | **1.18.1** |
| matplotlib | **2.0.2** |

### Notes
If you are on a mac, you may first install portaudio
> brew install portaudio

### Usage
> python stream.py
